Mini\+Copier is a graphical copy manager. It provides more comfort and control over file copy operations than basic OS functions. ~\newline
 You can use it to queue several transfers, resume a copy which has failed, limit the speed or pause the copy... ~\newline
 Mini\+Copier is multiplatform and supports Linux, Windows and Mac\+OS X systems.

\hypertarget{md__e_1__doxgen____project_minicopier-master_minicopier-master__r_e_a_d_m_e_autotoc_md2}{}\doxysection{Features}\label{md__e_1__doxgen____project_minicopier-master_minicopier-master__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item Pause a copy
\item Treatment of the transfers one after the other
\item Add new transfers to the queue while a copy is already being processed
\item Dynamic management of the queue of remaining transfers
\item Limit the speed of the copy
\item Skip current transfer to proceed to the next one
\item Resume a copy at the exact point where it failed (no need to start over)
\item Choose another name for the target if a file already exists
\item Can follow or ignore symbolic links (Unix systems only)
\item Set a default behavior if target file already exists
\item Keep track of the transfers which failed.
